Method
Algorithm
HOC is detected automatically — define any latent variable whose indicators include other LV names. ·  HOC =~ FOC1 + FOC2  ·  FOC1 =~ x1 + x2  ·  Two-stage always uses PLS for Stage 1.

Algorithm
Bootstrap
Include an interaction term using X*M notation in the structural path, e.g. Y ~ X + M + X*M. Bootstrap CIs are computed for βXM and simple slopes at −1 SD, mean, +1 SD.
Running moderation analysis…

Algorithm
Scale
IPMA plots each predictor LV by importance (total effect) on X and performance (rescaled composite mean) on Y. Provide the theoretical scale range for correct 0–100 rescaling; otherwise the observed range is used.
Running IPMA…
Specify the target LV and run to see the IPMA map here. Importance = total effect · Performance = rescaled composite mean (0–100).
Algorithm
Bootstrap
Use X*W in a structural path to specify moderation. a-path: M ~ X + W + X*W  ·  b-path: Y ~ M + W + M*W. Returns IMM (Index of Moderated Mediation) and conditional indirect effects at ±1 SD.
Running moderated mediation analysis…
Upload a dataset and run to see Moderated Mediation results here. IMM (Index of Moderated Mediation) + conditional indirect effects at −1 SD, mean, +1 SD.
Permutations
NCA tests necessary (not sufficient) conditions. CE-FDH (staircase) and CR-FDH (ceiling regression) lines are computed for every structural IV → DV pair. Effect size d: <0.1 negligible · 0.1–0.3 small · 0.3–0.5 medium · ≥0.5 large.
