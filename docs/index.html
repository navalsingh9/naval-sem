// capabilities

Everything you need for rigorous SEM

Built for academic research, consulting reports, and reproducible analysis pipelines.

📐

PLS-SEM & CB-SEM

Full PLS and CB algorithms with outer model estimation, inner model path coefficients, and iterative convergence — both in one app.

🔁

Bootstrapping

5000-sample bootstrap for t-statistics and confidence intervals on all path coefficients and loadings.

📊

Full Reliability & Validity

AVE, CR, Cronbach's α, HTMT, Fornell-Larcker — all computed and formatted in publication-ready tables.

🖥️

Visual Model Builder

Drag-and-drop canvas to build your structural model — no coding required. See paths and constructs visually before running.

📋

Report Export

Export complete results with all tables, model diagram, and fit indices — ready for your thesis or journal submission.

✈️

Fully Offline

Works entirely without an internet connection after install. No cloud, no account, no subscription — your data stays on your machine.

// quickstart

Up and running in minutes

Download, install, and run your first model — no coding needed.

WINDOWS 
# Option A — Installer (recommended)
# Download NAVAL-SEM-Setup.msi from GitHub Releases
# Double-click to install → Start Menu shortcut created

# Option B — Portable EXE (no install needed)
# Download NAVAL-SEM.exe → drop anywhere → double-click to run

# Note: Windows may show a SmartScreen warning on first run.
# Click "More info → Run anyway" — this is normal for unsigned apps.
MACOS 
# Download NAVAL-SEM.dmg from GitHub Releases
# Open the DMG → drag NAVAL-SEM.app to Applications

# First launch: right-click → Open (to bypass Gatekeeper)
# Subsequent launches: double-click as normal
LINUX (DEBIAN / UBUNTU) 
# Download the .deb from GitHub Releases, then:
sudo dpkg -i naval-sem_*.deb
naval-sem

# Or use the portable binary:
chmod +x NAVAL-SEM
./NAVAL-SEM
